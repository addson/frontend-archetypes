(em) - Unidade de medida relativa ao font-size atual do próprio elemento selecionado
ou relativa ao font-size do elemento herdado (body)

Texto no Body (20px ou 1em)

Container Externo mudando o font-size (1em)
Container Interno mudando o font-size (2em)

Container Externo mudando o width (10em)
Container Interno mudando o width (5em)

(rem) - Unidade de medida relativa ao font-size atual do elemento raiz (tag html)
O font-size da tag html é 30, logo...

Container Externo mudando o width (10 rem)
Container Interno mudando o width (5 rem)

(%) - Unidade de medida relativa ao tamanho da mesma propriedade no elemento pai

Container Externo mudando o width (50 % = 50% do width do body)
Container Interno mudando o width (50 % = 50% do width do div externo)

(vw) - Unidade de medida relativa a largura da nossa janela

Container Externo mudando o width (10 vw = 10% da largura da janela)
Container Interno mudando o width (5 vw = 5% da largura da janela)

(vh) - Unidade de medida relativa a largura da nossa janela

Container Externo mudando o width (10 vh = 10% da altura da janela)
Container Interno mudando o width (5 vh = 5% da altura da janela)

(vmax) - Unidade de medida relativa a maior dimensão da nossa janela

Container Externo mudando o width (10 vmax = 10% da maior dimensão da janela)
Container Interno mudando o width (5 vmax = 5% da maior dimensão da janela)

(vmin) - Unidade de medida relativa a maior dimensão da nossa janela

Container Externo mudando o width (10 vmin = 10% da menor dimensão da janela)
Container Interno mudando o width (5 vmin = 5% da menor dimensão da janela)
